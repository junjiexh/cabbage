# Cabbage

Cabbage is a web application, built with react. Golang for backend.

frontend: react + shadcn
backend: go + chi + postgre sql
devops: docker

Gemini api or other api to provide ai power.

I have set up frontend for you

## user operation flow

- User edit a todolist
- User push a 'plan my day' button
- backend generate a timeline and send back to frontend
- frontend render the timeline
